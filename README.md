# IFOME 

Nos últimos anos, a demanda por serviços de entrega de alimentos cresceu significativamente, impulsionada pela conveniência dos aplicativos de delivery e acelerada pela pandemia. Diante desse cenário, minha equipe e eu desenvolvemos o IFOME, um Sistema Integrado de Gestão de Entregas Gastronômicas (SIGEG), um projeto inspirado no modelo do iFood.

## Objetivo do Projeto

Criar uma plataforma eficiente e intuitiva que conecte restaurantes, entregadores e consumidores, oferecendo uma experiência otimizada e acessível para todos.

## Principais Funcionalidades

- Cadastro e Divulgação de Menus – Restaurantes podem se cadastrar e exibir seus menus detalhados com descrições, imagens e preços.

 - Integração com Sistemas de Pagamento – Suporte a plataformas como Stripe, Pagar.me e PagSeguro para transações seguras.

 - Rastreamento de Entregadores – Monitoramento em tempo real para estimativas precisas de entrega.

 - Gerenciamento de Pedidos – Acompanhamento do status do pedido com notificações automáticas.

 - Avaliação e Feedback – Usuários podem avaliar restaurantes e entregadores para melhorar a experiência.

 - Sistema de Recomendação – Sugestões personalizadas com base no histórico de pedidos.

 - Suporte ao Cliente – Canais dedicados para comunicação entre clientes, entregadores e restaurantes para resolver dúvidas e problemas rapidamente.

## Requisitos Não-Funcionais

Interface Intuitiva e Responsiva – Adaptável a computadores, tablets e smartphones.

Segurança e Privacidade dos Dados – Conformidade com regulamentações para proteção dos usuários.
